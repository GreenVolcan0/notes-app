О приложении

«Заметки» · версия 1.5.0

Простой менеджер заметок, написанный на чистом JavaScript в рамках практических работ №13–15 по дисциплине «Фронтенд и бэкенд разработка».

Что внутри:

  • Service Worker — перехват запросов и кэширование (практика 13).
  • Web App Manifest — установка как PWA (практика 14).
  • App Shell — мгновенная загрузка каркаса + динамический контент (практика 15).
  • HTTPS на localhost — через mkcert/openssl, обязательное условие для PWA.
  • localStorage — заметки переживают перезагрузку и офлайн.

Стратегии кэширования:

  • App Shell (HTML/JS/иконки): Cache First — сначала кэш, потом сеть.
  • Контент (/content/*): Network First — сначала сеть, при ошибке кэш, fallback на home.

Данная страница загружена через fetch('./content/about.html') и помещена в dynamic-content-v1. Попробуйте изменить этот текст, перезагрузить страницу — увидите свежую версию. Уйдите в офлайн — страница всё равно откроется (из dynamic cache).
